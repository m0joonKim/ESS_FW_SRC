# Embedded Storage System Projects

This repository contains the firmware source code for the **Embedded Storage System** course, developed on the **Cosmos+ OpenSSD** platform.

## Branches & Projects

### `main` (Project 1)
- **Topic**: Basic Firmware & Host Interface
- **Description**: Baseline firmware implementation focusing on the basic NVMe host interface and environment setup.

### `prj2_page_level` (Project 2 - baseline)
- **Topic**: FTL Mapping (Page-level)
- **Description**: Implementation of **Page-level Mapping** FTL.

### `prj2_block_level` (Project 2 - mjkim immpl)
- **Topic**: FTL Mapping (Block-level)
- **Description**: Implementation of **Block-level Mapping** FTL.

### `prj3_kvs` (Project 3)
- **Topic**: Key-Value SSD (KV-SSD) Implementation
- **Description**: Implementation of a Key-Value Store interface directly on the SSD firmware.
